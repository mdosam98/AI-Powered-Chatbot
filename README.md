# AI-Powered-Chatbot

AI-Powered Conversational Chatbot (Gemini LLM + Flask + Full-Stack)
A full-stack AI conversational assistant powered by Google Gemini LLM with intelligent multi-turn conversations, memory-based personalization, voice interaction, and a modern animated chat UI.
The system combines Machine Learning + API engineering + Full-Stack development to deliver a fast, human-like chatbot experience.
